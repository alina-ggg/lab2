## 📝 Описание заданий

### Задание №1: Минимальная подстрока
**Условие:** Даны две строки $S$ и $T$. Необходимо найти минимальную по длине подстроку $S$, которая содержит все символы $T$ с учётом их кратности.

**Пример:** $S = \text{"ADOBECODEBANC"}$, $T = \text{"ABC"}$ → Результат: $\text{"BANC"}$.

### Задание №2: Офис (Планирование встреч)
**Условие:** В офисе одна переговорная и $N$ заявок со временем начала и конца. Нужно составить расписание с максимальным количеством встреч без наложений. Время переводится в минуты от начала дня. Выводятся количество встреч и их порядковые номера.

**Пример:** $N=4$, заявки (540-600, 570-630, 600-660, 660-720) → Результат: 3 встречи, номера 1 3 4.

### Задание №3: Равенство предыдущему числу
**Условие:** На вход подаётся $N$ натуральных чисел. Необходимо определить, сколько чисел из этой последовательности равны своему предыдущему числу.

**Пример:** $N=4$, числа 12 12 33 33 → Результат: 2.
